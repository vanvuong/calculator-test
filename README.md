## Structure
```
project-name/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
    ...
  src/
    components
    containers
    store
        actions
        reducers
    utils
    index.js
    ...

```

## Enviroment
We will need NodeJS as develop enviroment.

## Start in development mode
To run the app in development mode, execute the command
```
npm install
npm run start
npm run test

```
